📖 Advanced Testing Guide

What Changed

Real Tokens: Fetches trending tokens from pump.fun and DexScreener APIs instead of random addresses. Also collects new tokens from the live stream.
Swap Flow Test: The critical production scenario—discovers new tokens and immediately monitors their trades.
Discovery→Trade Latency: Measures how long from token discovery to first trade event (critical for bot performance).
Working Charts: Bar charts now display real msg/sec data over time.
Message Details: See actual message previews, message types, sizes, and error categorization.
Subscription Rate: Slider controls how fast new tokens/wallets are subscribed to (respects rate limits).

Test Scenarios

🔄 Swap Flow (CRITICAL): Subscribe to new tokens → immediately subscribe to trades. Measures critical path latency.
New Tokens: Just the discovery stream—how many new tokens can you discover per second?
Trades (Real): Subscribe to real trending tokens and measure trade event throughput.
🐳 Whale: Paste wallet addresses to monitor their trades in real-time.
💪 Stress: Combines all three at max capacity.

Key Metrics

Throughput: Messages per second (msg/s)
Avg Message Size: Average bytes per message
Overhead: CPU/network utilization estimate
Discovery→Trade Latency: Milliseconds from token discovery to first trade
Conversion Rate: % of discovered tokens that receive at least one trade

Expected Results

Healthy: Overhead <50%, latency <500ms, errors <1%
Production Threshold: Overhead <70%, latency <1000ms
Capacity Found: Overhead hits 80%+, latency spikes >2000ms
